## Usage
```bash
deux dev [options]
```

## Options
`--db <path>` *Optional*  
Custom database path.

`--api` *Optional*  
Run in API Mode.

`--build` *Optional*  
Build all assets.

## CLI Example
```bash
# Default
deux dev

# API Mode
deux dev --build --api
```
