Switch to another theme built with `Deux-CLI`.

## Usage
```bash
deux switch [theme]
```

## Options
`--db <path>` *Optional*  
Custom database path.

`--list` *Optional*  
Print available themes.

`--api` *Optional*  
Run in API Mode.

## CLI Example
```bash
# Default
deux switch

# See theme list
deux switch --api --list
```
