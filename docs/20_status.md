Print current theme information from name, theme's URL, repository URL, total of templates, features, and more.

## Usage
```bash
deux status [options]
```

## Options
`--db <path>` *Optional*  
Custom database path.

`--api` *Optional*  
Run in API Mode.

## CLI Example
```bash
# Default
deux status

# API Mode
deux status --api
```
