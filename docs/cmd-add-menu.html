Add menu subcommand

This subcommand is part of deux add command.

Add and Register WordPress Navigation Menu. If menu use custom walker, there will class-menuname-nav-walker.php created in libraries directory.

Usage

deux add menu [options]

Options

--db <path> Optional
Custom database path.

--input <json> Optional
Set config in api mode without prompts.

--api Optional
Run in API Mode.

JSON Input

// JSON Example
{
  // @type Object
  // @required
  "menu": {
    // Menu name.
    // @type String
    // @required
    "name": "Primary",

    // Menu description.
    // @type String
    // @required
    "description": "Example Description",

    // Is use custom Menu Walker.
    // @type Boolean
    // @required
    "walker": true
  }
}

CLI Example

# Default
deux add menu

# API Mode
deux add menu --api --input '{ "menu": { "name": "Primary", "description": "Example Description", "walker": true } }'
